# Barotrauma-Extended-Medicine
A mod for a Barotrauma game which extends existing medicine with new medical items and afflictions.

While duplicating or/and creating new mods on basis of this, please leave references(like mention or link to github) to original author of mod. <br />
All files are allowed for shared use.

# Addons
If you want to modify or expand this mod in your own way, use addons folder.
Check README in there before editing.
